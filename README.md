# CPSC-334-project

## Description

A lightweight, interactive calculator that runs directly in the Linux terminal, built with Python and launched via a Bash script. Designed for developers and command-line users who want a fast, no-frills way to perform mathematical computations without leaving their terminal environment.

## Motivation to Choose This Project

To dive deeper into Continuous Integration and Continuous Development Practices, we chose to build this simple but useful application. The function of a calculator is to help solve math problems with many actions, such as: 
- Addition
- Subtraction
- Multiplication
- Division
- Exponents
- Logarithms
- And many more!


Through this project, we can each take on the simple tasks of one of these functions of a calculator at a time, and we can therefore integrate our code with each frequently (and ideally without many errors). 

We have an interest in math that makes coding these functions enjoyable and useful. We will learn about CI/CD together through group implementation. This calculator will serve as a great intro to these very useful topics.
